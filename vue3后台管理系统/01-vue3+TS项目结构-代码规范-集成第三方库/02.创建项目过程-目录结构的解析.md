## 创建项目

- 方式一：Vue CLI
  - 基于webpack工具
  - 命令行：vue create
- 方式二：create-vue
  - 基于vite工具
  - 命令：npm init vue@latest

```
// 创建细节
npm init vue@latest

Project name: 项目名
Add TypeScript? yes 要不要添加ts
Add JSX Support? no 要不要添加JSX,一般在react中用的多
Add Vue Router for Single Page Application development? no 是否添加路由
Add Pinia for state management? no 是否添加pinia状态管理工具
Add Vitest for Unit Testing? no 单元测试
Add Cypress for both Unit and End-to-End testting? no 端到端测试
Add ESLint for code quality? yes 代码规范
Add Prettier for code formatting? yes 代码保存自动格式化
```

## 目录结构

```
├─.vscode
│      extensions.json    // 最好在vscode里面安装一下里面推荐的插件
│      
├─public // 静态文件，打包时直接复制到/dist
│      favicon.ico
│      
└─src  // 源代码
    │  App.vue
    │  main.ts
    │  
    ├─assets
    │      base.css
    │      logo.svg
    │      main.css
    │      
    └─components             
│  .eslintrc.cjs
│  .gitignore // 提交git时，忽略一些文件
│  .prettierrc.json
│  env.d.ts // 放一些声明文件，比如为什么可以在.vue中使用ts语法。
│  index.html // 入口模板
│  package-lock.json // 锁定依赖最终的版本
│  package.json // 记录项目版本、名字，安装依赖
│  README.md // 文档
│  tsconfig.json // ts相关的配置文件
│  tsconfig.node.json
│  vite.config.ts // 配置文件，等同于vue.config.js
```
